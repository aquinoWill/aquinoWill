## 👨‍💻 About Me

I'm a Senior Frontend Engineer with **15+ years of experience** building and leading
high-performance web products for large-scale, international teams.

My craft lives at the intersection of **scalable frontend architecture**, **team
leadership**, and **AI-driven development workflows**. I specialize in React,
TypeScript, and Next.js, with a deep focus on performance (Core Web Vitals),
accessibility (a11y), and long-term code maintainability.

As a **Team Lead at Cyncly**, I drive technical direction, align cross-functional
teams, mentor engineers, and deliver complex features in a fully distributed,
international environment.

- 🏗️ Architecting scalable, maintainable frontend systems
- 🤖 Integrating AI coding tools (Cursor, Claude Code, GitHub Copilot) into daily dev workflows
- 🧪 Strong advocate for testing culture: Jest, React Testing Library, Cypress
- 🌐 Experience working with international teams across multiple time zones
- 📍 São Paulo, Brazil · Remote-first mindset
